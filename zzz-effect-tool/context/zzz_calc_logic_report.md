# zzz計算シート v1 計算ロジックまとめ
このExcelは主に **「ダメ計シミュテンプレ」**（汎用テンプレ）と **「シード-サンプル」**（具体例＋スキル倍率掛け）で計算しています。
## 1. シート構成
- **ダメ計シミュテンプレ**：入力（基礎値/ディスク/バフ/敵）→ 戦闘時ステータス → 3系統の「ダメージ基礎値（倍率を掛ける前）」を算出
- **シード-サンプル**：テンプレの式を流用しつつ、バフの期待値合算（例：0.6倍）や、スキル倍率を掛けた「個別スキルダメージ」を並べた例
- **ダメ計シートの使い方 / 最適化ツールの使い方 / 開発者の向け**：説明

## 3. 戦闘時ステータス/係数の計算（テンプレ）
テンプレは **「戦闘時ステータス（F列）」→「係数（I/L/O列）」→「ダメージ基礎値」**の順に流れます。

### 3.1 戦闘時ステータス（主にF列）
|セル|内容|式（抜粋）|
|---|---|---|
|F2|戦闘時 攻撃力|`= (B2 * (1 + B11*C11 + B23*C23 + B35) + B22*C22 + 316) * (1 + B38) + B37`|
|F3|戦闘時 HP|`= (B3 * (1 + B12*C12 + B25*C25 + B36) + B24*C24 + 2200) * (1 + B40) + B39`|
|F4|戦闘時 会心率|`= MIN(B4 + B15*C15 + B26*C26 + B43, 100%)`|
|F5|戦闘時 会心ダメ|`= B5 + B16*C16 + B27*C27 + B44`|
|F6|戦闘時 ダメバフ合計|`= B14*C14 + B45`|
|F7|戦闘時 異常マスタリ|`= B6 + B17*C17 + B29*C29 + B41`|
|F10|貫通率|`= B13*C13 + B47`|
|F11|貫通値|`= B28*C28`|
|F13|耐性ダウン|`= B48`|
|F14|ブレイク弱体倍率バフ|`= B52`|

### 3.2 係数とダメージ基礎値
|セル|内容|式（抜粋）|
|---|---|---|
|I6/L7|防御係数|`= MIN(794 / (794 + (F17 * (1 - F10) * (1 - F9) - F11)), 1)`|
|I7/O7|ダメージ耐性係数|`= 1 - F18 + F13  （※F18が敵耐性、F13が耐性ダウン）`|
|I8/L9/O8|ブレイク弱体倍率|`= F19 + F14`|
|I4/O4|会心係数（期待値）|`= 1 + 会心率*会心ダメ = 100% + F4*F5`|
|I12|通常スキル：ダメージ基礎値|`= I2*I3*I4*I5*I6*I7*I8*I9`|
|L12|異常：ダメージ基礎値|`= L2*L3*L4*L5*L6*L7*L8*L9*L10`|
|O12|透徹：ダメージ基礎値|`= O2*O3*O4*O5*O6*O7*O8*O9`|

## 6. 重要な注意点（ロジック上のクセ/改善案）
- **定数がセル化されていない**：`+316`（攻撃）や`+2200`（HP）、防御係数の`794`、レベル係数の`59/59`などが式に直書き。汎用化するなら別セルに出すと安全。
- **会心係数は期待値**：`1 + 会心率×会心ダメ` の形なので、会心が完全にランダム前提の平均ダメージになります（最大会心ダメージではない）。
- **ブレイク弱体倍率は加算**：`F19 + F14`（敵側＋バフ）になっているので、もし仕様が乗算ならここが変わります。
- **透徹は0除算対策あり**：`O3 = IFERROR(F12/O2, 0)` なので、変換効率が0のとき透徹計算は0に落ちます（シード-サンプルがその例）。
- **『最適化ツールの使い方』のセル番地が古い可能性**：説明ではターゲットセルが `I22` になっていますが、この版のテンプレではダメージ基礎値は `I12` にあります。
